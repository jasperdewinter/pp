Using a unique database of full-text daily newspaper articles spanning the period 1985-2018 from the largest financial newspaper in the Netherlands we use and a tone adjusted Bayesian mixture model, and use the extracted topics as predictors in a dynamic factor model. We extend the classical topic model in several respects. We contribute to the existing literature on topic models by introducing real-time estimation and hierarchy in the topic model structure. We conclude that the nowcasts of the dynamic factor model using the dynamic topics from the topic model alone can outperforms the same model using only monthly macro-economic indicators. Especially during volatile times such as the financial and the euro area sovereign debt crisis

Joint with Dorinth van Dijk (De Nederlandsche Bank).
